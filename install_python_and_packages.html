Install Python and Packages

1. install python 3

Install python 3.4 since some libraries has no 3.5 support.
Download here for x64 version.
Or go to here{:target=”_blank”} for other alternatives.

2. Add environment path to current user

  1. Open File Explorer or 文件资源管理器

  2. Right click This computer or 此电脑 and click Properties or 属性

  3. Add python 3 executables to path

    If you do not have PATH in your environmental variables create one.

3. Use pip to manage packages

  1. Open cmd console in Windows with administrator privilege.
