Install Python and Packages

1. install python 3

Install python 3.4 since some libraries has no 3.5 support.
Download here for x64 version.
Or go to here{:target=”_blank”} for other alternatives.

2. Add environment path to current user

  1. Open File Explorer or 文件资源管理器

  2. Right click This computer or 此电脑 and click Properties or 属性

  3. Add python 3 executables to path

    If you do not have PATH in your environmental variables create one.

3. Use pip to manage packages

  1. Open cmd console in Windows with administrator privilege.

  2. Execute C:\WINDOWS\system32>pip install pip --upgrade to upgrade pip

    e.g. C:\WINDOWS\system32>pip install pip --upgrade

  3. Install packages using pip

    e.g. C:\WINDOWS\system32>pip install pandas

  4. install packages from Unofficial Windows Binaries for Python Extension Packages

    Some python packages require compilation. To install them without compile them by yourself, you can download compiled packages from Unofficial Windows Binaries for Python Extension Packages.

    The files there are named under following naming convention.
    [package name]-[package version]-[python version]-[architecture].whl
    e.g. apsw-3.11.1.post1-cp34-cp34m-win_amd64.whl

    After downloaded it, you can install a .whl package by using pip
    e.g. C:\WINDOWS\system32>pip install apsw-3.11.1.post1-cp34-cp34m-win_amd64.whl
